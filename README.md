# GitTogether 🤝💻

GitTogether is a project-matching platform that helps students find collaborators to build real-world projects and gain hands-on experience.

It matches students based on skills, interests, learning goals, and availability, making it easier to turn ideas into shipped projects.

## Features
- Create a developer profile (skills, interests, experience level)
- Post or browse project ideas
- Match with collaborators who share your goals
- Real-time chat to start building immediately
- Contribution-based scoring system (DevScore)
- AI-powered project idea generator

## Problem We’re Solving
Students struggle to:
  - Find teammates for projects
  - Gain real-world experience
  - Discover what they actually enjoy building

GitTogether focuses on **building together**, not just networking.

## Solution

GitTogether connects students through:

  - Skill-based matching
  - Project idea discovery
  - Swipe-to-match system
  - Gamified experience levels

## Tech Stack
- Frontend: React + TypeScript + Vite
- Styling: Tailwind CSS
- Backend: Supabase
